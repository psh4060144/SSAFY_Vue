Vue 에서 데이터를 전달하는 방법

1. props
  상위 컴포넌트에서 하위 컴포넌트로 정보를 전달.
  아이의 미술시간을 위해 지점토를 사 주는 부모... 같은 느낌
  상위 컴포넌트에서 내려줄 때는 kebob-case, 하위 컴포넌트에서 받을 때는 camelCase 사용.
  왜? 내려주는 건 html 이고, 받는 건 js 잔슴... 엄빠가 밥해주고 아이는 낙타 가지고 노는거지 ㅋㅋㅋ 깔깔깔...

2. emit
  하위 컴포넌트에서 발생한 이벤트를 상위 컴포넌트에서 감지할 수 있도록 해 줌.
  아이가 가져온 가정통신문에 싸인을 하는 부모... 같은 느낌
